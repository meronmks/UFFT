# Unity製FFTソフト UFFT
Unityで高速フーリエ変換した結果を表示してくれるソフト

動作するものは下記boothにて配布中

https://meronmks.booth.pm/items/2864378

## Unityのバージョン
Unity 2020.3.13f1
